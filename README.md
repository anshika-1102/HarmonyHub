   ```bash

   cd HarmonyHub
   ```

2. **Set Up Local Server**  
   - If you're using XAMPP, move the project folder to the `htdocs` directory.
   - If you're using MAMP, place the project in the `MAMP/htdocs` folder.

3. **Start the Server**
   - Open XAMPP/MAMP and start Apache and MySQL services.

4. **Access the Application**
   - Open your browser and go to `http://localhost/HarmonyHub` (or the name of your project folder).

## âœ¨Features

- **Comprehensive Resource Hub**: Blogs, playlists, webinars, and workshops tailored to enhance mental well-being.
- **Self-Assessment Tests**: Inclusive tests to evaluate mental health conditions like depression, anxiety, ADHD, and PTSD.
- **Community Support**: WhatsApp and Discord groups for users to connect and share experiences.
- **Daily Positive Affirmations**: Encouraging messages to promote positivity and resilience.
- **Pomodoro Spiritual Breaks**: Mindful breaks to refresh and maintain productivity.
- **Diverse Activities**: Yoga, culturally appropriate exercises, and entertainment content like playlists and movies.
- **AI-Powered Chatbot**: 24/7 support offering personalized mental health advice.
- **Telemedicine Support**: Helplines and professional services for mental health.
- **Negative Thoughts Dustbin**: A symbolic way for users to dispose of negative thoughts.
- **Document Analysis Feature** (Upcoming): Analyze medical documents using the OpenText API.
